# Parkour Game

A third-person parkour platformer developed in Unreal Engine 5.
Navigate through floating low-poly environments using fluid movement mechanics 
built with C++ and Blueprints.

## Gameplay
- Third-person character controller with parkour mechanics
- Floating platform level design
- Low-poly art style with dynamic environment
- Character movement, jumping and collision systems

## Tech Stack
- Unreal Engine 5
- C++
- Blueprints
